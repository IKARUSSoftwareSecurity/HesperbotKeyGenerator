##Description
AndroidOS.Trojan.Hesperbot KeyGenerator

This is a quick hack tool for generating Activation and response (rCodes) for the AndroidOS.Trojan.Hesperbot
Android Trojan for testing. This code is part of the documentation that discusses the removal and cracking
of the same Trojan.

The current version of this key generator works
with the supposed "beta" version of Hesperbot, that surfaced in the April of 2014 in Austria.
